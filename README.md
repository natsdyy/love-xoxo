# 💖 Love, Xoxo | Luxury Management System

Welcome to the **Love, Xoxo** Business Management Suite. This is a premium, real-time administrative platform designed specifically for managing high-volume digital service subscriptions, partner earnings, and customer support lifecycles.

---

## 🌟 System Overview
**Love, Xoxo** provides a centralized hub for business owners and administrators to oversee every aspect of subscription sales. From the moment a stock item is added to the final salary payout, the system ensures precision, transparency, and efficiency.

### 🚀 Key Features
*   **Real-Time Analytics Dashboard**: Instantly track total revenue, profit, and sales performance across the entire team.
*   **Advanced Inventory (Stock Panel)**: Effortlessly manage digital accounts with categorized sorting (Entertainment, Educational, Editing, etc.) and slot-based monitoring.
*   **Automated Salary Engine**: A sophisticated commission system that automatically calculates partner earnings based on dynamic profit tiers.
*   **Transaction Lifecycle Tracking**: Full visibility into pending reservations, approved sales, refunds, and replacements.
*   **Supplier Order Management**: Specialized tools for owners to track and manage incoming stock from external suppliers.

---

## 🛠️ The Technology Stack
We use a state-of-the-art tech stack to ensure the system is fast, secure, and always in sync.

*   **Core Framework**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/) - Ensuring a lightning-fast user interface and smooth navigation.
*   **Database & Security**: [Google Firebase](https://firebase.google.com/) - Provides military-grade security and real-time data synchronization. When an admin makes a sale, the owner sees it instantly.
*   **Styling & Design**: [Tailwind CSS](https://tailwindcss.com/) - A custom-crafted "Luxury Aesthetic" with smooth transitions, glassmorphism effects, and a premium color palette.
*   **Icons & Visuals**: [Lucide React](https://lucide.dev/) - Clean, modern iconography for an intuitive user experience.

---

## 👥 Role-Based Access
The system is divided into two distinct access levels to maintain security and focus:

### 👑 Business Owner (`owner`)
*   **Full Financial Control**: Access to capital costs, price management, and overall profit analytics.
*   **Team Oversight**: View salary breakdowns for all admins and manage user accounts.
*   **Supplier Tracking**: Manage the `Orders` section to track incoming bulk stocks.

### 🛡️ Administrator (`admin`)
*   **Sales Focus**: Record new sales, manage pending customer requests, and monitor active account slots.
*   **Personal Performance**: View individual sales history and real-time salary calculations.
*   **Support Tools**: Handle account replacements and refund requests efficiently.

## 🚦 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   NPM or Yarn

### Installation
1.  **Clone the repository**:
    ```bash
    git clone [repository-url]
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run in Development Mode**:
    ```bash
    npm run dev
    ```

---

## 🔐 Security Note
This system uses **Firestore Security Rules** to ensure that only authorized members can access business data. 

> [!IMPORTANT]
> Always ensure you have applied the latest rules from `FIRESTORE_RULES.txt` in your Firebase Console to prevent unauthorized access.
